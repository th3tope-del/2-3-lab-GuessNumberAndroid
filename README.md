Интерфейсы компьютерных и мобильных систем

Тема работы: Основные этапы разработки Android-приложения
Название приложения: «Угадай число»

Цель работы
Изучить основные этапы разработки Android-приложений в среде Android Studio, освоить создание пользовательского интерфейса и реализацию логики приложения на языке Java.

Используемые технологии
Android Studio
Язык программирования Java
XML для описания пользовательского интерфейса

Описание приложения
Приложение «Угадай число» загадывает случайное число в диапазоне от 0 до 100. Пользователь вводит число в поле ввода и нажимает кнопку. Приложение сообщает, больше или меньше введённое число по сравнению с загаданным. В случае совпадения выводится сообщение об успешном угадывании и предлагается начать игру заново.

Структура проекта
MainActivity.java — файл, содержащий логику работы приложения
activity_main.xml — файл разметки пользовательского интерфейса

Результат работы
В результате выполнения лабораторной работы было разработано Android-приложение «Угадай число», которое корректно запускается и работает на эмуляторе Android.
